Kulinaria website disusun agar dapat menyebarluaskan AR card yang dapat diunduh dimanapun dan kapanpun.

Kulinaria.apk adalah aplikasi AR yang dibuat dengan menggunakan Unity.

Algoritma:
1. User membuka website,
2. Mengeksplorasi makanan/kuliner khas Indonesia,
3. User mengunduh Kulinaria APP yang merupakan aplikasi AR yang telah dibuat di mobile phone,
    diunduh dengan mengklik button pada page #more,
4. User mengunduh AR card dari page #discover yang digunakan untuk mentrigger 3D view AR,
5. User memiliki pengalaman menggunakan AR sekaligus mengenal kuliner otentik khas Indonesia.
